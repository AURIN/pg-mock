# pg-mock


Mock-up of a PostgreSQL database in Node.js (it emulates the `pg` library)


## Pre-requireemnts

* Node.js 10.16.x
* NPM 6.9.x
* Mocha (installed glaablly): 2.5.x


## Installation

```bash
npm run install
```


## Unit test

```bash
npm run utest
```


## Use

Have a look at `test/unit.js` for examples.
